# IonicQRScanner
QRScanner build on Ionic platform.


#### Dependencies
- Install Cordova Barcode plugin

``` $ cordova plugin add https://github.com/wildabeast/BarcodeScanner.git ```

- Install ngCordova

``` $ bower install ngCordova ```
